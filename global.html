MenuBar(activeView, onNavigate) → {React.ReactElement}

MenuBar Component (Props-Based) Displays navigation options and calls the onNavigate callback when a menu option is clicked. The parent component (App.jsx) manages the active view state and passes it down via the activeView prop. This makes data flow explicit: Menu clicks → onNavigate callback → parent state update
Parameters:
Name Type Description
activeView string The currently active view name (determines which button is highlighted)
onNavigate function Callback function called with the view name when a menu option is clicked
